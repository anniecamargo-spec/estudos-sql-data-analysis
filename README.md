📊 Análise de Dados Populacionais com SQL

Este projeto demonstra a aplicação de consultas SQL para manipulação e agregação de dados demográficos brasileiros.

O que foi desenvolvido:

Cruzamento de dados entre tabelas (INNER JOIN) utilizando chaves primárias e estrangeiras.

Consolidação de valores numéricos por categoria através da função SUM e da cláusula GROUP BY.

Organização de resultados para análise comparativa (ORDER BY).

Ferramentas: SQLite / DBeaver.
